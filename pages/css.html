O que é CSS?

Cascading Style Sheets (CSS): podemos traduzir como Folhas de Estilo em Cascata. É no CSS que definimos o visual da página. Criamos o estilo da página por meio de regras, que são aplicadas em cascata.

O que é possível fazer com CSS?

  • Alterar a fonte, cor e tamanho do texto.
  • Alterar o tamanho e cor de um bloco.
  • Definir margens e bordas.
  • Posicionar os blocos da página.

Criar Regras no CSS

Uma regra possui duas partes: o seletor e a declaração.

- Seletor: define em quais elementos a regra será aplicada.

- Declaração: define o visual.

Exemplo:

  • p: é o seletor. Indica que todos os elementos <p> (Parágrafo) do HTML serão selecionados para aplicar a regra.
  • {}: as propriedades devem estar dentro das chaves.
  • font-size: é o nome de uma propriedade. Ela altera o tamanho da fonte.
  • 20px: é o valor da propriedade. Neste caso, 20 pixels.

Seletor de Classe

O seletor de classe seleciona todos os elementos que possuem o atributo HTML “class” com um determinado valor. Esse atributo é global e é utilizado para aplicar o CSS.

Exemplo de Código HTML

        
            <p>Parágrafo normal</p>
            <p class="destacado">Parágrafo destacado</p>
            <p>Parágrafo normal</p>
            <p class="destacado">Parágrafo destacado</p>

Hierarquia no seletor

Também é possível definir um seletor que procure um elemento dentro de outro elemento.

Sintaxes:

– nome1 nome2: seleciona descendente de “nome1”.

– nome1 > nome2: selecionar “nome2” que é filho em primeiro nível de “nome1”.

Exemplo de Código HTML

        
            <p>Parágrafo dentro do corpo.</p>
            <section>
              <p>Parágrafo dentro da seção</p>
              <article>
                 <p>Parágrafo dentro do artigo</p>
              </article>
            </section>

Usando: section > p apenas o parágrafo dentro da seção será aplicado o estilo

Usando: section p será aplicado o estilo no parágrafo dentro do artigo e dentro da seção.

Propriedades de Texto em CSS

Cores de Texto

Em HTML são utilizados os nomes das cores em inglês ou valores hexadecimais para definir uma cor.

Exemplos de cores em RGB (Red Green Blue): vermelho: #FF0000, verde: #00FF00, azul: #0000FF, branco: #FFFFFF, preto: #000000

Tamanho do Texto em CSS

Em CSS, muitas propriedades alteram tamanhos. Salvo em algumas situações, sempre é necessário explicitar qual a unidade de medida utilizada. A mais conhecida é o pixel

Unidade em: Essa unidade define um tamanho relativo ao tamanho da fonte padrão do navegador (geralmento definidade em 16px).

Exemplo de uso: .maior { font-size: 2.5em }
