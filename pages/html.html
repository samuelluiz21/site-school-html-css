O que é HTML?

HTML(HyperText Markup Language) é uma Linguagem de Marcação de HiperTexto responsável em exibir conteúdo na Web.

O HTML usa "Marcação" para para exibir conteúdos em um navegador da Web.

A marcação HTML utiliza o sinal de menor ( < ) o nome do elemento e o sinal de maior (> ), assim: . Essas marcação são chamadas de "tags ".

O que são Tags?

As "Tags " são o conjunto de caracteres usados para formar um elemento HTML. Exemplo: Quando mencionamos a tag "p " estamos referenciado estes caracateres: <p>

Existem 2 tipos de Tags:

  • Tags que não necessitam de fechamento. Exemplos: <br> <hr> <meta> <img>
  • Tags que necessitam de fechamento. Exemplos: <p> </p> <html> </html> <body> </body>

Estrutura Básica de um documento HTML

Um arquivo HTML deve possuir uma estrutura básica de elementos, composta pelas tags: composta pelas tags: <!DOCTYPE< <html> <head< <body>. Veja a seguir uma estrutura básica de um documento HTML. 

        
            <!DOCTYPE html>
            <html>
                <head>
                    <title>Título</title>
                </head>
                <body>
                    Código html
                </body>
            </html>

Principais Tags HTML

Títulos

Quando queremos indicar que um texto é um título em nossa página, utilizamos as tags de heading em sua marcação:

Exemplo do uso do Código:

        
            <h1> Titulo Principal </h1>
            <h2> Titulo Secundario </h2>

Resultado:

Titulo Principal

Titulo Principal

Parágrafos

Para Exibir qualquer texto em nossa página

Exemplo do uso do código:

        
            <p> Primeiro Parágrafo </p>
            <p> Segundo Parágrafo </p>

Resultado:

Primeiro Parágrafo

Segundo Parágrafo

Imagens

Para incluir uma imagem, utiliza-se a tag <img>. Esta tag não deve ser fechada. Seus atributos obrigatórios são: "src": do inglês “source” (fonte), que é o caminho da imagem e o "alt": texto alternativo, que serve para quando não puder exibira imagem.

Exemplo de uso do código:

        
            <img src="img/logo.utfpr.png" alt="Logo da UTFPR">
